✅ Завдання 1. Ієрархія інтерфейсів
Створіть інтерфейс IShape з методом Draw().
Створіть інтерфейс IColoredShape, який успадковується від IShape і додає метод SetColor(string color).
Створіть клас Circle, який реалізує IColoredShape.
Завдання: створіть список об’єктів типу IShape, де зберігаються кола з різними кольорами, і виведіть їх опис.
